## Project Overview
This project presents an interactive Power BI dashboard built using OLA ride booking data. The dashboard helps analyze business performance, customer behavior, and operational efficiency through dynamic visualizations and KPIs.

## Key Insights
- Total Bookings
- Revenue Analysis
- Ride Status Analysis
- Cancellation Analysis
- Customer Ratings
- Driver Ratings
- Vehicle Type Performance
- Payment Method Analysis
- Booking Trends
- Distance Analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV Dataset

## Skills Demonstrated
- Data Cleaning
- Data Transformation
- Data Modeling
- DAX Measures
- Interactive Dashboard Design
- Business Intelligence
- KPI Reporting

## Dashboard Features
- Interactive slicers
- Dynamic KPI cards
- Trend analysis
- Drill-down reports
- Business insights
